# Smart-Farmland-Management-System-using-Spring-Cloud-and-Vue
智慧农田管理平台主要分为农场主页系统公告通知、农场统计分析、农田管理、农作物资管理、系统管理五大业务模块，全平台嵌入通用智能问答功能，为所有操作人员提供农事、业务咨询支撑。平台使用者分为管理员和农场员工两类，系统依据业务范围与操作权限完成角色划分： 管理员：拥有系统全部功能操作权限，可完成系统自定义配置、全模块数据管理，同时可使用智能问答查询系统运维、权限配置相关操作指引； 农场员工：按岗位职责细分采购人员、销售人员、农田管理人员，仅开放对应岗位业务模块，可通过智能问答咨询作物种植、农资调度、产销规划等各类农事业务问题。
